# ALi Spy Pro V2.5

> E-commerce Spy Tool for Arab Markets (COD)

## Features
- Facebook Ad Library scraping for Saudi Arabia, Kuwait, UAE, Morocco, Iraq
- TikTok Top Ads analysis
- COD product research with keyword filtering
- Break-even calculator with AI analysis (Gemini)
- Multi-market comparison
- Arabic RTL interface

## Tech Stack
- React + TypeScript
- Tailwind CSS
- Gemini AI API
- Firebase (optional backend)

## Setup
```bash
npm install
npm run dev
```

## Environment Variables
```
VITE_GEMINI_API_KEY=your_key_here
```
